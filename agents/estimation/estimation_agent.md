# Estimation Agent

## Purpose

The Estimation Agent is responsible for analyzing Jira user stories and providing Agile story point estimates based on technical complexity, effort, and implementation considerations.

---

## Responsibilities

- Retrieve Jira story details
- Understand functional requirements
- Analyze implementation complexity
- Evaluate technical effort
- Consider development impact
- Consider testing effort
- Identify dependencies
- Provide story point estimation
- Explain estimation rationale
